# Skyward Rest - Narrative Design Portfolio Piece

A narrative-driven technical demo developed in Unreal Engine 5. This project showcases the integration of branching storytelling, interactive systems, and environmental narrative techniques.

## Key Features

* Branching Dialogue System: Built using Data Tables for easy content management and scalability.
* Narrative Logic: Implemented complex dialogue flows and quest triggers using Blueprints.
* Interactive Environments: Integrated item inspection and world-state changes based on player choices.
* Level Design: Created an atmospheric environment to support the "Show, Don't Tell" narrative philosophy.

##  Technical Stack

* Engine: Unreal Engine 5.4+
* Scripting: Blueprints Visual Scripting
* Data Handling: Data Tables & Structs
* Version Control: Git LFS
